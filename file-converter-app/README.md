# File Converter App

A lightweight web application for converting between various file formats.

## Features
- Document conversion: PDF ↔ Word ↔ TXT ↔ HTML
- Image conversion: JPG/PNG/WebP/SVG with compression
- Audio/Video conversion: MP4/AVI/MOV and MP3/WAV
- Spreadsheet conversion: Excel ↔ CSV ↔ JSON
- Code formatting for various programming languages

## Tech Stack
- Backend: Python Flask
- Frontend: Vue.js + Tailwind CSS
- Conversion libraries: python-docx, PyPDF2, Pillow, pandas, ffmpeg-python, etc.
