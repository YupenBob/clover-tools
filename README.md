## 🚀 快速上手

**在线使用（推荐）**
直接访问：https://tools.xsanye.cn
无需注册，打开即用。所有工具纯前端实现，无需上传数据，隐私安全。

**部署到自己的服务器**
```bash
git clone https://github.com/YupenBob/clover-tools.git
cd clover-tools
npx serve .
```

**本地开发**
```bash
git clone https://github.com/YupenBob/clover-tools.git
npm install
npx serve .
```
