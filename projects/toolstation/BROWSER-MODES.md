## 📊 模式对比

| 特性 | 无头模式 (Headless) | 有头模式 (Headful) |
|------|-------------------|------------------|
| **显示界面** | ❌ 无界面 | ✅ 有窗口界面 |
| **资源消耗** | ⚡ 较低 | 💪 较高 |
| **检测风险** | ⚠️ 可能被识别 | ✅ 正常浏览器 |
| **网站兼容性** | ⚠️ 部分网站限制 | ✅ 完全兼容 |
| **调试便利性** | ❌ 需要日志 | ✅ 可视化查看 |
| **适用场景** | 本地测试/CI | 爬虫/自动化 |

## 🔧 当前配置

```bash
  --no-sandbox \

  --window-size=1920,1080
```

```bash

  --no-sandbox \
  --remote-debugging-port=9222 \
  --window-size=1920,1080 \
  --start-maximized
```

## 🛠️ 配置命令

### 切换到无头模式
```bash
# 停止当前浏览器
pkill -f chromium

# 启动无头浏览器
chromium-browser \
  --no-sandbox \
  --headless \
  --remote-debugging-port=9222 \
  --window-size=1920,1080 \
  > /tmp/chromium.log 2>&1 &
```

### 切换到有头模式
```bash
# 停止当前浏览器
pkill -f chromium

# 启动 Xvfb
export DISPLAY=:99
Xvfb :99 -screen 0 1920x1080x24 > /tmp/xvfb.log 2>&1 &

# 启动有头浏览器
chromium-browser \
  --no-sandbox \
  --remote-debugging-port=9222 \
  --window-size=1920,1080 \
  --start-maximized \
  > /tmp/chromium.log 2>&1 &
```
