## 🔔 Cron 配置

在 `openclaw.json` 中添加：

```json
{
  "cron": {
    "entries": {
      "daily-morning": {
        "enabled": true,
        "schedule": "0 6 * * *",
        "command": "system event --text \"🌅 早间检查：查看昨晚进度，今天计划完成哪些工具？\" --mode now"
      },
      "daily-noon": {
        "enabled": true,
        "schedule": "0 12 * * *",
        "command": "system event --text \"☀️ 午间检查：上午完成了多少？下午继续加油！\" --mode now"
      },
      "daily-evening": {
        "enabled": true,
        "schedule": "0 18 * * *",
        "command": "system event --text \"🌙 晚间检查：今天完成了哪些工具？更新 PROGRESS.md\" --mode now"
      }
    }
  }
}
```
